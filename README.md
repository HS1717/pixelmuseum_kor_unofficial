# pixelmuseum_kor_unofficial
Unofficial korean translation of "Juufuutei Raden™'s Guide for Pixel Museum"
주우후테이 라덴이 안내하는 픽셀 뮤지엄의 비공식 유저 한국어 패치입니다.
해당 패치는 스팀판만 적용 가능하며, 스위치판은 적용 불가능합니다.
